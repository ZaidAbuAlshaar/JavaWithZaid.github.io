الأسئلة الشائعة

ما هي أنواع البيانات الأساسية في جافا؟

أنواع البيانات الأساسية في جافا تشمل:

  • الأعداد الصحيحة (int): مثل 5, 10, -3
  • الأعداد العشرية (double): مثل 3.14, 2.5, -0.5
  • السلاسل النصية(الكلمات) (String): مثل "مرحباً", "جافا"
  • القيم المنطقية (boolean): true أو false
int age = 25;
double height = 1.75;
String name = "زيد";
boolean isStudent = true;

كيف يمكنني أخذ مدخلات من المستخدم؟

يمكنك استخدام الكلاس Scanner لأخذ مدخلات من المستخدم: 

import java.util.Scanner;

public class UserInputExample {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        
        System.out.print("أدخل اسمك: ");
        String name = scanner.nextLine();
        
        System.out.print("أدخل عمرك: ");
        int age = scanner.nextInt();
        
        System.out.println("مرحباً، " + name + "! عمرك " + age + " سنة.");
    }
}

ما الفرق بين المصفوفات (Arrays) والقوائم (Lists)؟

الاختلافات الرئيسية:

  • المصفوفات (Arrays): حجم ثابت ولا يمكن تغيير حجمها بعد التهيئة
  • القوائم (Lists): حجم ديناميكي ويمكن إضافة/حذف العناصر بسهولة
// مصفوفة
String[] myArray = {"تفاح", "موز", "كرز"};
myArray[1] = "توت";

// قائمة
List<String> myList = new ArrayList<>();
myList.add("تفاح");
myList.add("موز");
myList.add("كرز");
myList.set(1, "توت");
myList.add("برتقال");

المستوى 1: أنواع البيانات والإدخال

شرح المستوى

في هذا المستوى، ستتعلم أساسيات التعامل مع البيانات في جافا. سنتعرف على الأنواع الرئيسية مثل الأعداد الصحيحة (int)، الأعداد العشرية (double)، السلاسل النصية (String)، والقيم المنطقية (boolean). الأهم من ذلك، ستتعلم كيفية جعل برنامجك يتفاعل مع المستخدم عبر استقبال المدخلات باستخدام `Scanner`.

المفاهيم الرئيسية

  • أنواع البيانات الأساسية في جافا
  • الفرق между int و double
  • كيفية التعامل مع النصوص (String)
  • القيم المنطقية (true/false)
  • استخدام Scanner لقراءة المدخلات

أمثلة التعليمات البرمجية

import java.util.Scanner;

public class DataTypesExample {
    public static void main(String[] args) {
        // تعريف متغيرات بأنواع مختلفة
        int age = 25;
        double height = 1.75;
        String name = "زيد";
        boolean isStudent = true;
        
        // طباعة القيم
        System.out.println("الاسم: " + name);
        System.out.println("العمر: " + age);
        System.out.println("الطول: " + height);
        System.out.println("هل هو طالب؟ " + isStudent);
        
        // قراءة المدخلات من المستخدم
        Scanner scanner = new Scanner(System.in);
        System.out.print("أدخل اسمك: ");
        String userName = scanner.nextLine();
        System.out.println("مرحبًا " + userName + "!");
    }
}
