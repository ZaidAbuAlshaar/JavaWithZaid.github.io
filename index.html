أتقن جافا مع زيد

تعلم لغة جافا من الأساسيات إلى المفاهيم المتقدمة مع دروس تفاعلية واختبارات مصممة خصيصًا لك 

public class HelloWorld {
    public static void main(String[] args) {
        // طباعة رسالة ترحيبية
        System.out.println("مرحبًا بك في عالم جافا!");
        
        // حساب بسيط
        int x = 5;
        int y = 10;
        int sum = x + y;
        
        System.out.println("مجموع العددين هو: " + sum);
    }
}

ما هي أنواع البيانات الأساسية في جافا؟

أنواع البيانات الأساسية في جافا تشمل:

  • الأعداد الصحيحة (int): مثل 5, 10, -3
  • الأعداد العشرية (double): مثل 3.14, 2.5, -0.5
  • السلاسل النصية(الكلمات) (String): مثل "مرحباً", "جافا"
  • القيم المنطقية (boolean): true أو false
int age = 25;
double height = 1.75;
String name = "زيد";
boolean isStudent = true;

كيف يمكنني أخذ مدخلات من المستخدم؟

يمكنك استخدام الكلاس Scanner لأخذ مدخلات من المستخدم: 

import java.util.Scanner;

public class UserInputExample {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        
        System.out.print("أدخل اسمك: ");
        String name = scanner.nextLine();
        
        System.out.print("أدخل عمرك: ");
        int age = scanner.nextInt();
        
        System.out.println("مرحباً، " + name + "! عمرك " + age + " سنة.");
    }
}

ما الفرق بين المصفوفات (Arrays) والقوائم (Lists)؟

الاختلافات الرئيسية:

  • المصفوفات (Arrays): حجم ثابت ولا يمكن تغيير حجمها بعد التهيئة
  • القوائم (Lists): حجم ديناميكي ويمكن إضافة/حذف العناصر بسهولة
// مصفوفة
String[] myArray = {"تفاح", "موز", "كرز"};
myArray[1] = "توت";

// قائمة
List<String> myList = new ArrayList<>();
myList.add("تفاح");
myList.add("موز");
myList.add("كرز");
myList.set(1, "توت");
myList.add("برتقال");

ما هي أهم مبادئ البرمجة كائنية التوجه (OOP) في جافا؟

أهم مبادئ البرمجة كائنية التوجه في جافا هي:

  • التغليف (Encapsulation): إخفاء تفاصيل التنفيذ الداخلية
  • الوراثة (Inheritance): إعادة استخدام الكود من خلال إنشاء علاقات بين الفئات
  • تعدد الأشكال (Polymorphism): إمكانية تنفيذ نفس العملية بطرق مختلفة
  • التجريد (Abstraction): إخفاء التفاصيل المعقدة وإظهار الوظائف الأساسية فقط

كيف يمكنني التعامل مع الملفات في جافا؟

يمكن استخدام FileReader و FileWriter أو المكتبات الحديثة مثل java.nio.file: 

// قراءة ملف باستخدام Files
try {
    List<String> lines = Files.readAllLines(Paths.get("ملف.txt"));
    for (String line : lines) {
        System.out.println(line);
    }
} catch (IOException e) {
    System.out.println("خطأ في قراءة الملف: " + e.getMessage());
}

// كتابة ملف
try {
    Files.write(Paths.get("ملف.txt"), 
               Arrays.asList("سطر 1", "سطر 2"), 
               StandardOpenOption.CREATE);
} catch (IOException e) {
    System.out.println("خطأ في كتابة الملف: " + e.getMessage());
}

ما هي الاستثناءات (Exceptions) وكيف نتعامل معها في جافا؟

الاستثناءات هي أحداث غير متوقعة تحدث أثناء تنفيذ البرنامج. يمكن التعامل معها باستخدام:

  • try-catch: للتعامل مع الاستثناءات عند حدوثها
  • throws: لنقل مسؤولية التعامل مع الاستثناء للدالة المستدعية
  • finally: لتنفيذ كود بغض النظر عن حدوث استثناء أم لا
try {
    // كود قد يسبب استثناء
    int result = 10 / 0;
} catch (ArithmeticException e) {
    System.out.println("خطأ في القسمة: " + e.getMessage());
} finally {
    System.out.println("هذا الكود سينفذ دائماً");
}
